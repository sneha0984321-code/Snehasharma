# 🕵️‍♀️ Mini Project: Ethical Hacking – Footprinting & Reconnaissance

## 📖 Overview
This project demonstrates **footprinting and reconnaissance techniques** used in ethical hacking.  
The goal is to gather information about a target system or organization using **Open Source Intelligence (OSINT)** methods, without causing harm.  
It highlights both **basic** and **advanced** approaches to reconnaissance.

---

## 🔍 Techniques Covered

### Basic Footprinting
- **Social Media Research** – Collecting public information from platforms like Facebook, Instagram, X.
- **Search Engine Hacking** – Using tools such as **Shodan** and **Censys** to analyze devices/networks via IP addresses.
- **Google Dorks** – Advanced queries (`inurl:`, `filetype:`, `intitle:`) to uncover hidden or sensitive data.

### Advanced Footprinting
- **Technology Analysis** – Tools like **Netcraft** and **Wappalyzer** reveal backend technologies and missing security features.
- **Subdomain Discovery** – Identifying subdomains with Pentest tools or Subdomain Finder.
- **URL Extraction** – Using link extractors to list all site URLs (faculty, careers, fees, etc.).
- **Buffer Size Identification** – Understanding packet handling capacity, relevant for DoS attack analysis.
- **Security Headers Check** – Using **securityheaders.com** to detect weak/missing headers.
- **Website Time Travel** – Leveraging the **Wayback Machine** to analyze historical versions of the site.

---

## 🎯 Key Takeaways
- Footprinting is the **first step in ethical hacking**, helping map the environment and identify weaknesses.
- It reduces effort in later hacking phases and exposes sensitive data leaks.
- Demonstrates both **basic reconnaissance** (social media, search engines) and **advanced methods** (subdomain discovery, buffer size analysis, security headers).
